# weather-app

A modern weather app built with HTML, CSS, and JavaScript that fetches real-time weather data using an API and displays temperature, humidity, and wind speed.

🚀 Features

🌍 Search weather by city name

🌡️ Displays temperature

💧 Shows humidity

🌬️ Wind speed information

☁️ Dynamic weather icons

⚡ Real-time API data

🛠️ Technologies Used

1.HTML

2.CSS

3.JavaScript

4.Weather API

🔧 How to Use

1.Clone the repository

2.Open index.html in your browser

3.Enter a city name

4.View the weather details instantly
